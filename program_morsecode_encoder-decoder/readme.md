# Morse Code Encoder

A simple C++ program that converts text into **Morse code** and saves the encoded output to a `.txt` file.

---

## Features
- Converts alphabets (A–Z, case-insensitive) into Morse code.
- Converts digits (0–9) into Morse code.
- Represents spaces between words as `/`.
- Saves encoded Morse code to `morsecode.txt`.

---

## How It Works
1. Enter any text when prompted.
2. The program encodes it into Morse code.
3. The encoded result is displayed in the console and also saved to a file named `morsecode.txt`.
