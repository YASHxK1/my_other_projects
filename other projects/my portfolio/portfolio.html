Connecticut Property Sale Price Predictor

This project is an interactive tool designed to predict property sale amounts in Connecticut based on town, property type, and listing year. Using historical property sales data, it leverages machine learning to provide accurate price estimates. Users can select a town, property type, and year from dropdown menus and sliders to instantly see the predicted sale amount. Additionally, the tool displays historical averages and trends for context, making it useful for buyers, sellers, and real estate professionals.

Technologies: Python,Pandas & NumPy,Scikit-learn,IPywidgets
